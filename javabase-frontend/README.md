# javabase-frontend

Frontend do [JavaBase](../README.md) — React 18 + TypeScript + Vite + Tailwind v4.

```bash
npm install
npm run dev      # http://localhost:5173 — precisa do backend rodando em :8080
npm run build    # tsc -b && vite build
```

Ver o README principal do projeto na raiz do repositório para arquitetura, endpoints e stack completa.
